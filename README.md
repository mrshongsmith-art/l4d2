# l4d2

`l4d2` 是一个用于 AstrBot 的《Left 4 Dead 2》服务器管理插件，支持查询服务器状态、切换地图、管理地图文件、添加地图下载任务和解析 Steam 创意工坊链接。

## 功能特性

- 查询 L4D2 服务器在线状态、当前地图、人数、延迟和在线玩家列表。
- 在群聊中输出服务器连接命令。
- 支持官方地图和三方地图切换。
- 支持查看、删除、重命名和清空 Web 管理器中的地图列表。
- 支持通过 URL 添加地图下载任务。
- 支持从群文件上传地图到 Web 管理器。
- 支持解析 Steam 创意工坊链接，并自动加入下载任务。
- 支持按群和管理员 QQ 配置权限。

## 安装方式

将本插件目录复制到 AstrBot 插件目录中：

```text
data/plugins/l4d2
```

安装依赖：

```bash
pip install -r requirements.txt
```

然后重启 AstrBot，或在 AstrBot 控制台中重新加载插件。

## 配置说明

插件优先使用 AstrBot 面板配置，也会读取插件目录下的 `config.json`。

首次配置时可以参考 `config.example.json`，复制为 `config.json` 后填写自己的服务器信息：

```json
{
  "webManager": {
    "baseUrl": "http://127.0.0.1:58641",
    "username": "",
    "password": ""
  },
  "serverAddress": "127.0.0.1:27015",
  "groupIds": ["123456789"],
  "adminUsers": ["10000"]
}
```

配置项说明：

- `webManager.baseUrl`：L4D2 Web 管理器地址。
- `webManager.username`：Web 管理器账号，按你的管理器实现填写。
- `webManager.password`：Web 管理器令牌或密码。
- `serverAddress`：L4D2 服务器地址，格式通常为 `IP:端口`。
- `groupIds`：允许响应插件命令的群号列表。
- `adminUsers`：允许执行切图、地图管理、下载等管理命令的 QQ 号列表。

请不要把包含真实账号、密码、令牌、服务器管理地址的 `config.json` 上传到公开仓库。

## 常用命令

- `求生帮助`：查看插件帮助。
- `查询` / `查服` / `综合查询`：查询服务器状态。
- `切换官方图`：查看官方地图列表。
- `切换官方图2`：切换到官方地图列表中的第 2 个地图。
- `切换地图` / `切换三方图`：查看三方地图列表。
- `切换地图1-2`：切换到第 1 个三方图战役的第 2 个小图。
- `地图` / `地图列表`：查看 Web 管理器中的地图列表。
- `删除地图2`：删除编号为 2 的地图。
- `删除地图2-6`：删除编号 2 到 6 的地图。
- `重命名地图2 新名称`：重命名指定编号的地图。
- `清空地图`：清空地图列表。
- `上传地图 文件名`：从群文件中按文件名添加地图。
- `地图下载 <url>` / `下载地图 <url>`：添加地图下载任务。
- `下载任务`：查看当前下载任务。
- `创意工坊 <链接>` / `创意工坊解析 <链接>`：解析 Steam 创意工坊链接并添加下载任务。

## 依赖

- `python-a2s`
- `python-valve`
- `aiohttp`

## 使用注意

- 地图管理、下载、上传和切换地图等操作需要配置 `adminUsers`。
- 服务器查询需要目标 L4D2 服务器可被当前 AstrBot 运行环境访问。
- Web 管理器接口、令牌和权限配置会直接影响地图管理功能是否可用。
- 本仓库提供的是插件代码，不包含你的私有服务器配置。
